Embedded Rust on ESP32

A self-directed learning journey for Year 11 Digital Solutions. Start here to build the prerequisite knowledge before moving into local setup and FIA2 simulation work.

Why Rust? Introduction to Embedded Systems

Understand what embedded systems are, why Rust is used for reliable low-level programming, and how ESP32 boards fit into Digital Solutions prototyping.

  • Define embedded systems and microcontrollers
  • Compare Rust with Python for classroom prototyping
  • Identify FIA2 links to programmed solutions
L2
60 min

Setting Up Your Forge

Install the Rust toolchain, editor support, and ESP32 target requirements. Confirm your machine can compile a basic program successfully.

  • Install Rust and Cargo
  • Discuss editor workflow and extensions
  • Run a first successful build
L3
60 min

Rust Fundamentals for Embedded Work

Explore variables, types, functions, conditionals, loops, and ownership at a beginner-friendly level, with examples connected to sensor-based logic.

  • Read and modify simple Rust code
  • Use conditionals and loops for control logic
  • Relate program structure to hardware interaction
L4
60 min

GPIO Control & Project Readiness

Learn the purpose of GPIO pins, practise simple input/output concepts, and prepare to move into the simulation lesson with confidence.

  • Explain what GPIO means
  • Map simple sensor/actuator relationships
  • Prepare for the FIA2 simulation workflow
