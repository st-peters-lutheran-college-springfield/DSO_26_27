Unit 2: Data Analysis and UX Principles

Analyse data and apply user experience principles in real-world digital contexts

Unit Overview

In Unit 2, students will learn to analyse data using computational thinking and apply user experience (UX) principles to design digital solutions that meet user needs.

Key Topics:

  • Data collection and analysis techniques
  • UX design principles and user research
  • Wireframing and prototyping
  • Usability testing and iteration

Learning Outcomes:

By the end of this unit, you will be able to analyse real-world data, create user personas, design wireframes, and test digital solutions with users.
