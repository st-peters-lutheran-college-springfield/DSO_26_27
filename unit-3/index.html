Unit 3: Data-Driven Prototypes

Develop and evaluate data-driven prototypes with a focus on quality and testing

Unit Overview

In Unit 3, students will develop functional prototypes using data-driven design principles, with emphasis on quality assurance, testing methodologies, and iterative improvement.

Key Topics:

  • Prototype development frameworks
  • Data integration and API usage
  • Quality assurance and testing strategies
  • Performance optimization
  • Iterative design and feedback loops

Learning Outcomes:

By the end of this unit, you will be able to build functional prototypes, integrate data sources, conduct testing, and refine solutions based on evidence.
