Unit 1: Creating with Code

General Subject Digital Solutions (QLD) · Learning resources for the 2026 / 2027 course

Unit Focus

  • Computational and design thinking foundations
  • Designing and developing digital solutions with code
  • Evaluating the quality and impact of solutions

Learning Goals

  • Read, write, test and improve code with purpose
  • Use data and logic to solve authentic problems
  • Document decisions and reflect on outcomes

Suggested Learning Sequence

  • Getting started with code structure and syntax
  • Using variables, selection and iteration effectively
  • Building and refining a small digital solution

Assessment Preparation

  • Interpret the task criteria and evidence requirements
  • Track implementation iterations and testing outcomes
  • Communicate design choices with justified reasoning
