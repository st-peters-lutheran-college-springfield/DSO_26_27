Unit 1: Creating with Code

General Subject Digital Solutions (QLD) · Learning resources for the 2026 / 2027 course

Assessment preparation focus: build strong problem solving, justified design choices, testing evidence and clear communication so your FIA2 response is original, rubric aligned and well supported.

Unit Focus

  • Computational and design thinking foundations
  • Designing and developing digital solutions with code
  • Evaluating the quality and impact of solutions

Learning Goals

  • Read, write, test and improve code with purpose
  • Use data and logic to solve authentic problems
  • Document decisions and reflect on outcomes

Suggested Learning Sequence

  • 1
    Getting started with code structure and syntax
    Build confidence with basic program structure, readable syntax, and simple computational thinking patterns.
  • 2
    Using variables, selection and iteration effectively
    Develop stronger logic by applying variables, conditional statements, loops, and testable problem-solving steps.
  • 3
    Building and refining a small digital solution
    Create, test, and improve a small solution while documenting design choices and evidence of refinement.
  • 4
    Simulation-supported FIA2 preparation
    Use Wokwi to prototype, test, and document programmed solutions before deploying to physical hardware.
  • 5
    Optional Rust pathway
    Students exploring Rust should complete the flipped-classroom prerequisite lesson before attempting the Rust simulation workflow.
Open FIA2 Simulation Lesson Open Rust Flipped Classroom Hub
Rust pathway prerequisite: Students choosing Rust as their coding language should complete the Rust on ESP32 — Flipped Classroom Hub before attempting the Rust simulation activities in Wokwi.

Assessment Preparation

  • Interpret the task criteria and evidence requirements
  • Track implementation iterations and testing outcomes
  • Communicate design choices with justified reasoning
